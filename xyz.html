C Code: Syntax, Semantic, and Logical Errors

#include <stdio.h>

int main() {
    int a = 10;
    int b = 0;

    // Syntax Error: Missing semicolon
    printf("Hello, World!") // Error: expected ';' before '}' token

    // Semantic Error: Division by zero
    int c = a / b; // Error: division by zero

    // Logical Error: Incorrect calculation
    int d = a * 2; // Error: should be 'a + 2' instead of 'a * 2'

    printf("Result: %d\n", d);
    return 0;
}
  • Syntax Error: Missing semicolon after printf.
  • Semantic Error: Dividing a number by zero.
  • Logical Error: Multiplication used where addition was intended.
